# DROPAMIN BRAND IDENTITY & DESIGN SPECIFICATIONS

> Updated according to the pivot on 2026-05-12.

## 1. Brand Vision
Dropamin is a Shuffle Dance Academy that embodies **Radosť** (Joy). It is modern, energetic, premium, and authentic. It feels like "the scene" — club culture meets professional dance education.

## 2. Color Palette
| Element | Hex Code | Description |
| :--- | :--- | :--- |
| **Pure Black** | `#000000` | Primary background, contrast, stability. |
| **Pure White** | `#FFFFFF` | Text readability, clean accents. |
| **Amethyst Purple** | `#6300E2` | Creativity, club culture, premium vibe. |
| **Lime Green** | `#CCFF00` | Freshness, energy, high-contrast action. |

### 2.1 The Signature Gradient
**Amethyst Purple (`#6300E2`) → Lime Green (`#CCFF00`) → Pure Black (`#000000`)**
Use for: Dynamic backgrounds, scroll animations, and key visual bridges.

## 3. Typography
- **Body Text**: **Inter**
  - Use for: General copy, descriptions, navigation.
  - Vibe: Clean, modern, highly readable.
- **Accents & Design Elements**: **Space Mono**
  - Use for: Highlights, technical specs, "hacker/tech" vibe details, specific headings.
  - Vibe: Technological, electronic music culture.

## 5. Page Structure (from landing-page.md)
1. **Hero**: Headline: *"Drop the beat, feel the dopamine."* Dynamic animation/video.
2. **About**: Focus on Community, Vibe, Movement, and Dopamine.
3. **Levels**: Beginner, Intermediate, Advanced, Fanatic. "Age doesn't matter, experience does."
4. **Why Shuffle**: Kardio (500-850 kcal), stress reduction, community.
5. **Community**: Social proof. Rating: **9.47/10** (5/5 ⭐ on Google).
6. **Pricing/Registration**: Clear CTA to Linktree or form.
7. **Footer**: Contact (Matúš Vršanský), Socials, Žilina location.

## 6. Tone & Voice (Voice of the brand)
- **Authentic, Energetic, Confident.**
- Communicate "from the scene" (tutor to dancer), not "corporate."
- Use "Ty" (informal singular).
- **Prohibited:** "kvalitné kurzy", "super komunita", "zábava pre každého". (See `zakazy.md`).

## 7. Technical Requirements
- **Mobile-first** is the absolute priority.
- **Dark UI** is the primary mode.
- High-contrast elements using Lime Green on Black.
